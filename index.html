Home Page

Welcome to the SkiPro! official website. Here, you can view what can SkiPro! do for you! People love skiing, one of the popular winter sports. However, you can ski only in winter and if you're unlucky, you may injure yourself! Luckily, SkiPro! brings skiing to the world of a computer game that will keep your entertained and injury-free for entire year!

Software Features

SkiPro! is built using C++ and SFML (Super Fast Multimedia Library) for Windows platform. The game is a RPG-like, since the tiling is used to make the background. And sprites serve for obstacles and players. The map and resources are in separate files included in the program. Also, several SFML DLLs are included, since the SFML-code can't run without them. Use keyboard to navigate player past the obstacles. You have 3 lives. Each hit costs you a life. Zero lives = Loss. In future, the gameplay may be further expanded.

FAQs

1. The Game won't start. What should I do? The game contains several dlls and resources (images and file text). Without them, the program won't start. If problem occurs, try downloading the program again. 2. Can I modify the files (names and content)? Please don't. The game depends on them, and any alteration to the files may cause erratic behaviour of the game.
